# 📈 AI Stock Prediction Platform

A professional-grade, AI-powered stock market analysis and prediction platform. This application combines **Deep Learning (LSTM)** with **Real-time News Sentiment Analysis** to provide traders with comprehensive insights, technical indicators, and automated trade signals.

## 🚀 Key Features

### 1. AI-Driven Predictions
- **LSTM Deep Learning**: Uses a Long Short-Term Memory (LSTM) neural network trained on 15 years of historical data to forecast future price movements.
- **Sentiment Weighting**: Integrates real-time news headlines via NewsAPI and performs sentiment analysis using TextBlob to adjust AI forecasts based on market mood.

### 2. Comprehensive Dashboard
- **Live Ticker Tape**: Real-time price updates for trending stocks and indices.
- **Dynamic Charts**: Interactive visualizations of actual vs. predicted prices.
- **Key Metrics**: Instantly view Market Cap, P/E Ratio, 52-Week High/Low, and more.

### 3. Advanced Analysis Tools
- **Technical Indicators**: View RSI, MACD, and Bollinger Bands with custom timeframes (1M to 5Y).
- **Financial Statements**: Access full Income Statements, Balance Sheets, and Cash Flow data directly via Yahoo Finance.
- **Sector Performance**: Monitor YTD performance across Tech, EV, Finance, Healthcare, and Energy sectors.

### 4. Security & Authentication
- **Secure Login/Signup**: Features OTP-based email verification for new accounts.
- **Admin Panel**: Dedicated dashboard for user management and broadcast notifications.
- **Environment Security**: All sensitive API keys and secrets are managed via `.env` files.

## ⚙️ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/ashukr03/Stock-Prediction-App.git
cd Stock-Prediction-App
```

### 2. Install Dependencies
It is recommended to use a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Configure Environment Variables
Create a `.env` file in the root directory and add your keys:
```env
FLASK_SECRET_KEY=your_flask_secret
NEWS_API_KEY=your_news_api_key
GEMINI_API_KEY=your_gemini_api_key
SMTP_SENDER=your_email@gmail.com
SMTP_PASSWORD=your_app_password
```

### 4. Run the Application
```bash
python flask_app.py
```
Open your browser and navigate to `http://127.0.0.1:5000`.
